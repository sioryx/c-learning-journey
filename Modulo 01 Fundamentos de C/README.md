# Módulo 01 - Fundamentos de C

## Descripción

Este módulo reúne los primeros ejercicios desarrollados durante mi aprendizaje del lenguaje C.

El objetivo principal fue construir una base sólida en programación estructurada, comprendiendo la sintaxis del lenguaje, el uso de variables, operadores, estructuras de control y ciclos antes de avanzar hacia temas de mayor complejidad.

Cada ejercicio fue realizado de manera incremental, permitiéndome practicar repetidamente los conceptos fundamentales hasta dominarlos.

---

# Objetivos del módulo

- Familiarizarme con la sintaxis del lenguaje C.
- Comprender el flujo de ejecución de un programa.
- Utilizar correctamente variables y tipos de datos.
- Dominar operadores aritméticos, relacionales y lógicos.
- Implementar estructuras condicionales.
- Resolver problemas mediante ciclos.
- Desarrollar la lógica necesaria para construir programas cada vez más complejos.

---

# Conceptos aprendidos

Durante este módulo se estudiaron los siguientes temas:

- Estructura básica de un programa en C
- Entrada y salida estándar
- Variables
- Tipos de datos
- Operadores aritméticos
- Operador módulo
- Operadores de incremento y decremento
- Operadores de asignación
- Operadores relacionales
- Operadores lógicos
- Condicionales (`if`, `else if`, `else`)
- Sentencia `switch`
- Ciclos `while`
- Ciclos `do while`
- Ciclos `for`

---

Al finalizar este módulo fui capaz de:

- Tomar decisiones utilizando estructuras condicionales.

# Ejercicios destacados

Algunos ejercicios fueron especialmente importantes desde el punto de vista didáctico:

- **021_Mayor_De_Tres_Numeros**  
  Primer ejercicio que combina múltiples condiciones utilizando operadores lógicos.

- **030_Menu_De_Combate_Do_While**  
  Simula un pequeño menú interactivo utilizando un ciclo `do while`, representando uno de los primeros programas con un flujo más cercano a una aplicación real.

- **031–034 Examen Final**  
  Serie de ejercicios integradores donde se combinaron variables, operadores, estructuras condicionales y ciclos para resolver problemas completos.
