C LEARNING JOURNEY 
Bienvenido a mi C Learning Journey.
Este repositorio documenta mi proceso de aprendizaje del lenguaje C,
desde los conceptos mas basicos hasta el desarrollo de proyectos cada vez mas complejos.

El objetino no es únicamente almacenar ejercicios, sino registar mi evolcuión como programador
y construir una base sólida para el desarrloo de software y videojuegos.
Cada módulo representa una etapa de este recorrido y cada commit refleja un nuevo aprendizaje.

OBJETIVOS DEL PROYECTO
*Aprender lenguahe C desde sus fundamentos.
*Desarrollar una lógica de programación sólida.
*Aplicar buenas prácticas de programación.
*Dominar Git y Github como herramienta de control de versiones.
*Construir una biblioteca personal de funciones reutilizables.
*Prepararme para proyectos de mayor complejidad.
