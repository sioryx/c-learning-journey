# Índice - Módulo 04: Organización y Tipos Definidos por el Usuario

Este módulo introduce la organización de proyectos en C mediante estructuras, tipos definidos por el usuario y modularización, culminando con el desarrollo de un sistema de combate organizado en múltiples archivos.

---

## Contenido

### 001. Primer Struct
Introducción a las estructuras (`struct`) para representar entidades complejas.

### 002. Validación Typedef
Uso de `typedef` para simplificar la declaración de tipos personalizados.

### 003. Enumeraciones
Definición y utilización de enumeraciones (`enum`) para representar conjuntos de valores constantes.

### 004. Implementación de Módulos
Separación del proyecto en archivos fuente (`.c`) y archivos de cabecera (`.h`).

### 005. Integración Módulo Equipo
Integración de módulos independientes para construir un sistema funcional.

### 006. Arrays Estructuras
Manipulación y recorrido de arreglos de estructuras.

### 007. Gestión de Equipos
Implementación de funciones para administrar equipos de personajes mediante punteros y arreglos.

### 008. Combate Por Equipos
Desarrollo de la lógica básica de combate entre equipos utilizando estructuras y módulos.

### 009. Proyecto Integrador Sistema De Combate
Integración de todos los conceptos del módulo en un proyecto modular completo.

---

## Resumen del módulo

- Estructuras (`struct`)
- `typedef`
- Enumeraciones (`enum`)
- Modularización
- Archivos `.h` y `.c`
- Separación de responsabilidades
- Arreglos de estructuras
- Punteros a estructuras
- Gestión de equipos
- Proyecto integrador
