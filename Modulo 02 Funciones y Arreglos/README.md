# Módulo 02 - Funciones y Arreglos

## Introducción

Este módulo marca la transición desde los programas secuenciales hacia el desarrollo de soluciones modulares y reutilizables. A lo largo de los ejercicios se introducen las funciones como mecanismo para dividir un problema en tareas específicas y los arreglos como la primera estructura de datos capaz de almacenar múltiples elementos del mismo tipo.

Posteriormente, ambos conceptos se integran para construir algoritmos más completos, donde las funciones reciben, procesan y modifican arreglos. Esta evolución permite desarrollar programas más organizados, reutilizables y fáciles de mantener.

Todos los ejercicios fueron desarrollados en lenguaje C y posteriormente organizados en este repositorio utilizando una estructura profesional, documentando el progreso del aprendizaje y manteniendo un historial claro mediante Git y GitHub.

---

# Objetivos

* Comprender la declaración, definición e invocación de funciones.
* Utilizar parámetros y valores de retorno.
* Organizar programas mediante funciones especializadas.
* Comprender el funcionamiento de los arreglos unidimensionales.
* Recorrer, modificar y procesar arreglos mediante ciclos.
* Implementar algoritmos básicos sobre arreglos.
* Integrar funciones y arreglos para construir soluciones reutilizables.
* Desarrollar código con una estructura más clara y modular.

# Conclusiones

Este módulo representa un punto de inflexión en el aprendizaje del lenguaje C.

Se abandona la programación basada exclusivamente en instrucciones secuenciales para comenzar a construir programas organizados mediante funciones reutilizables y estructuras de datos.

Las bases adquiridas aquí servirán como fundamento para módulos posteriores dedicados a punteros, cadenas, estructuras, archivos y memoria dinámica, donde estos conceptos serán utilizados de forma constante.

La documentación de este módulo refleja no solo los ejercicios realizados, sino también la evolución del proceso de aprendizaje y la adopción gradual de buenas prácticas de programación y control de versiones mediante Git y GitHub.
